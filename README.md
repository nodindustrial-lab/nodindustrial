# NOD INDUSTRIAL — Web V3 (multi-página)

## Qué incluye
- `index.html` (Inicio)
- `about.html` (Si quieres saber más)
- `services.html` (Qué hacemos)
- `sectors.html` (Sectores)
- `contact.html` (Contacto)
- `styles.css` (estilos)
- `assets/` (imágenes)

## Importante: sustituir imágenes
En `assets/` hay ficheros placeholder (texto). Cámbialos por tus fotos, manteniendo los nombres:

- `assets/hero.jpg`
- `assets/pci.jpg`
- `assets/pv.jpg`
- `assets/intralog.jpg`
- `assets/team.jpg`

Recomendación: comprime y ajusta a 1600–2000 px de ancho para carga rápida.

## GitHub Pages
- Settings → Pages → Deploy from a branch → `main` + `/(root)` → Save
- URL típica: `https://TU-USUARIO.github.io/TU-REPO/`

Si el repo se llama `nodindustrial`, la home es `index.html` en la raíz.
